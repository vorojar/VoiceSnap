==========================================================================
                      VoiceSnap 语闪 - 极速离线语音输入
==========================================================================

感谢您选择 VoiceSnap 语闪！
这是一款基于 FunASR 和 Sherpa-Onnx 构建的高性能离线语音输入软件。
它完全运行在本地，使用时无需联网，保护您的隐私，并且响应速度极快。

[核心特性]
1. ⚡ 极速响应：基于 C# 原生开发，启动仅需 0.1 秒，内存占用低。
2. 🔒 完全离线：内置 SenseVoice 高精度大模型，无需联网，数据不出本地。
3. 🎯 精准识别：支持中、英、日、韩、粤语混合识别，自动添加标点。
4. 🎈 极简交互：独特的"长按说话，松手即输"交互模式，不打断工作流。

[安装与运行]
1. 确保您的电脑运行的是 Windows 10 或 Windows 11。
2. 直接双击运行目录下的 "VoiceSnap.exe"。
3. 首次运行可能需要几秒钟下载及加载模型，请耐心等待。
4. 成功启动后，屏幕右下角会出现一个半透明的胶囊指示器，显示"按住Ctrl说话"。

[使用方法]
1. 【长按说话】：在任意可输入文字的地方（如微信、Word、浏览器），按住键盘左下角的【Ctrl】键不放。
2. 【语音输入】：指示器变红并显示波形时，开始说话。
3. 【松手即输】：说完后松开 Ctrl 键，文字将自动打字上屏。

[设置说明]
- 托盘图标：在系统托盘区（右下角时间旁）可以找到 VoiceSnap 的图标。
- 右键菜单：右键点击托盘图标可以退出软件。
- 左键点击：左键点击托盘图标可以打开设置面板，您可以在此：
  - 修改触发按键（支持 Ctrl, Alt, Shift, CapsLock 等）。
  - 设置开机自启。
  - 开启/关闭自动隐藏指示器。

[常见问题]
Q: 为什么按住 Ctrl 没有反应？
A: 请检查指示器状态是否为"按住Ctrl说话"。如果是"加载中"，请稍等。如果软件未启动，请检查托盘区是否有图标。另外，请确保没有其他软件占用了 Ctrl 键的全局功能（虽然很少见）。

Q: 识别准确率不高？
A: 请尽量使用清晰的普通话，并靠近麦克风。软件默认使用系统默认录音设备，请在系统声音设置中确认默认麦克风是否正确。

Q: 杀毒软件误报？
A: 由于软件涉及全局按键监听（用于实现长按说话）和模拟键盘输入，可能会被部分杀毒软件误判。本软件完全开源且安全，请放心添加信任。
