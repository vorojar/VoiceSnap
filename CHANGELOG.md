# Changelog

## v2.3.0 (2026-02-27)

### 新功能

- **Windows 7 专用版** — 新增 `VoiceSnapWin7` 精简版，支持 64 位 Windows 7
  - 去除 Wails/WebView2 依赖，使用原生 Win32 系统托盘
  - CPU 推理（DirectML 需 Win10，Win7 不支持）
  - 使用 Go 1.20（最后支持 Win7 的版本）+ onnxruntime-for-win7 兼容层
  - 首次启动自动下载语音模型（~152 MB），带 Win32 原生进度条
  - 托盘右键菜单：设置热键、提示音开关、开机自启动、GitHub 链接、退出
  - 热键设置对话框：按键捕获，支持 Ctrl/Alt/Shift/CapsLock/F1-F12
  - 7z 自解压打包（`build_sfx.sh`），最终发布文件仅 6.7 MB
  - 91 个 DLL 兼容层（含 Win7 API shim），解压即用

## v2.1.1 (2026-02-27)

### Bug 修复

- **设置开关闪烁错误值**：开机自启动、音效反馈、自动隐藏等开关在后端数据加载完成前不渲染，避免先显示默认值再跳变
- **开机自启动开关重启后显示未开启**：开关区域加 loading 守卫，后端返回真实状态后才渲染
- **历史保留天数重启后回退默认**：保留天数选择器同样加 loading 守卫
- **热键显示为"Ctrl"而非"R-Ctrl"**：启动时从后端加载真实热键名称，默认右 Ctrl 正确显示为 R-Ctrl

## v2.1.0 (2026-02-26)

### Bug 修复

- **开机自启动开关不记忆**：重启后开关状态从后端（注册表）加载，不再恢复默认值
- **历史保留时长不记忆**：修复"永久保留"(0天) 重启后回退为 30 天的问题
- **设置项初始值不同步**：音效反馈、自动隐藏开关也从后端加载真实状态
- **F10/F11/F12 快捷键名称错误**：F10 显示为 "F:" 等乱码，改用 Sprintf 正确显示

### 新功能

- **语气词自动过滤**：识别结果自动去除"嗯、啊、呃、额、哦"等中文语气词和 um/uh/hmm 等英文填充词
- **自由对讲静音自动停止**：连续 3 秒无声自动结束录音并识别（含 2 秒开头保护期），无需手动按键停止
- **胶囊指示器 DPI 自适应**：根据系统 DPI 缩放胶囊大小、字体、动画，高分屏不再显示过小

## v2.0.0 (2026-02-15) — Go 跨平台重写

VoiceSnap 2.0 使用 Go + Wails v3 + Svelte 从零重写，目标是更小体积、更优体验、未来跨平台。

### 重写亮点

- **全新技术栈**：从 C# WPF/.NET 8 迁移至 Go + Wails v3 + Svelte 5
- **体积缩减 57%**：运行文件从 ~80 MB 降至 ~34 MB（不含模型）
- **内存优化**：~400 MB（含模型），相比 WPF 版 ~500 MB
- **Apple Design UI**：全新设计语言，白色卡片 + #f2f2f7 分组背景 + SVG 图标
- **原生浮动指示器**：Win32 GDI+ Layered Window，无焦点抢夺

### 新功能

- **两种输入模式**：长按说话 + 短按自由说话（WPF 版仅长按）
- **录音音效反馈**：马林巴风格双音提示（开始升调、完成降调），可在设置中关闭
- **录音时长显示**：指示器实时显示录音计时（0:00 格式），长按和自由说话模式均支持
- **Escape 取消录音**：录音中按 Esc 立即取消，比组合键更直觉
- **剪贴板保护**：粘贴前保存旧剪贴板内容，粘贴后自动恢复
- **识别历史**：自动保存最近 50 条识别结果，支持复制、单条删除、清除所有，可设置保留时长（7/30/90天/永久）
- **麦克风设备选择**：下拉菜单列出所有输入设备，点击切换，选择自动保存
- **中英混合优化**：识别后自动在中英文/数字之间插入空格，句首字母自动大写，清理 SenseVoice 特殊标记
- **指示器位置记忆**：浮动指示器支持鼠标拖拽，位置自动保存到配置文件，下次启动恢复
- **隐私声明**：关于页展示离线/无需注册/开源三条隐私保证，让用户安心
- **中英双语界面**：自动跟随系统语言，托盘菜单同步本地化
- **自定义快捷键**：支持 Ctrl / Alt / Shift / 功能键 / 字母键 / 空格等

### 技术细节

| 模块 | 实现 |
|---|---|
| 音频录制 | malgo (miniaudio)，16kHz/16-bit/mono PCM |
| 语音识别 | sherpa-onnx SenseVoice，DirectML GPU → CPU 自动回退 |
| 全局热键 | GetAsyncKeyState 30ms 轮询，支持长按/短按/组合键过滤 |
| 文字粘贴 | Win32 剪贴板 + SendInput Ctrl+V，失败回退 Unicode SendInput |
| 浮动指示器 | Win32 GDI+ Layered Window，WS_EX_NOACTIVATE + TOPMOST，可拖拽 + 位置记忆 |
| 文本后处理 | CJK/ASCII 边界自动加空格，句首大写，特殊标记清理 |
| 音效合成 | Go 程序化生成 WAV (48kHz, 基频+谐波+指数衰减)，winmm.dll PlaySound |
| 识别历史 | JSON 文件持久化，最多 50 条，按保留天数自动清理 |
| 前端 | Svelte 5 + Vite 6 + TypeScript，~80 KB 产物 |
| 配置 | JSON 格式，兼容 WPF 版 config.json |
